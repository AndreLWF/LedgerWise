# LedgerWise
Full-stack fintech app built with React Native (Expo), FastAPI (Python), and Supabase. Connects to real bank accounts via Teller API for transactions, balances, and account management. Free-tier architecture: Railway (API), Vercel (web), Cloudflare R2 (storage), Upstash Redis (cache). Designed to scale from friends &amp; family to production.
